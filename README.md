## Инструкции
### Разворачивание системы
1. Установите файлы проекта в директорию.
2. Измените `.env` файл по примеру в `.env_example`
3. Запустить Docker Compose: `docker compose up -d --build`.
4. Скрипт сразу начнет свою работу.
